# Standardized Agent Prompts

This directory contains reusable prompts and instructions to guide AI coding assistants (like Antigravity) through common tasks.

## Prompts Registry (Planned)
- `create_implementation_plan.md`: Instructions on how to draft clear technical plans for user review.
- `create_pull_request.md`: Prompt structure for summarizing code diffs and creating clear PR descriptions.
- `debug_accounting_anomaly.md`: Guidance on diagnosing FIFO lot and pool valuation errors.

*Note: Over time, we will expand this directory with specific prompts based on patterns we discover during development.*
